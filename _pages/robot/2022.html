Programming

This year, we decided to write our robot code in Java, as opposed to the Pyhton we have used previously. In addition to our robot code, we developed a dashboard for our touchscreen driver station. It is written in JavaScript/HTML/CSS, and communicates with the robot through a python-based tornado server. It is based on the FRCDashboard framework we developed in 2016.

We drive using a pilot-copilot operating system, with one driver controlling robot motion and the other controlling all other robot functions, such as the shooter. This allows each driver to control their own independent task, making operating the robot more efficient.
